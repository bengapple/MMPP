<h3>Description</h3>
This is an R-implementation of Markov Modulated Poisson Process for unsupervised event detection in time series of counts data.

<h3>Requirements</h3>
This requires the base installation of R and the following libraries:
<ul>
<li>expm
<li>MCMCPack
</ul>
